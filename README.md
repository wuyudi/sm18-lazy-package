Below is the origin README:

# SuperMemo 18 中文懒人包

该项目旨在提供一个开箱即用的 SuperMemo 18 程序，内置中文补丁，以及汉化教程。

## 文件介绍

- sm18 —— SuperMemo 18 本体 + 懒人包教程，打开 sm18.exe 即可使用
- 字体 —— 内置 方正悠宋 和 Times Ten LT Std 两个字体
- Quicker —— 内置了推荐新手使用的 sm18 quicker 动作列表
